수업자료로 제공한 9개의 물체 중 케이크, 모자, 자동차1, 2, 칼, 집을 선택하였다.
케이크는 여러 개의 케이크를 특정한 점을 중심으로 회전운동하는 것을 구현했다.
모자와 집은 배경에서 천천히 올라가는 움직임을 생성했다.
칼은 여러 개가 윈도우 창을 중심으로 방향키로 회전운동을 하게 프로그램을 짰다.
자동차 2개는 충돌하여서 하나의 자동차는 임의의 방향으로 튕겨나가는 움직임을 구사했다.

곰 얼굴을 새로 모델링하였고, 위의 케이크와 비슷하게 회전운동하는 것을 구사했다.
